A professional, futuristic VPN application with **native implementations** for Android (Kotlin) and Windows (C#).

## ✨ Features

### 🔐 VPN Connection System
- ✅ Real VPN connection management
- ✅ Animated connect/disconnect interface
- ✅ Live connection timer with digital display
- ✅ Real-time upload/download speed monitoring
- ✅ Auto-reconnect capability
- ✅ Kill switch protection
- ✅ DNS protection & leakage prevention
- ✅ Connection status tracking

### 🌍 Server Management
- ✅ Multiple premium server locations
- ✅ Ghana network optimization (MTN, Telecel, AT)
- ✅ Real-time ping speed display
- ✅ Favorite server bookmarking
- ✅ One-tap connection
- ✅ Smart server selection

### 🔒 Security & Privacy
- ✅ No-logs policy
- ✅ Fingerprint/PIN lock (Android)
- ✅ App protection (Windows)
- ✅ Encrypted local storage
- ✅ Secure VPN tunnel

### 🎨 Premium UI/UX
- ✅ Futuristic dark neon purple theme
- ✅ Glassmorphism card design
- ✅ Cyberpunk-inspired dashboard
- ✅ Smooth animations & transitions
- ✅ Premium gradients
- ✅ Real-time statistics display

## 📱 Platform Support

### Android
- **Minimum SDK**: Android 8.0 (API 26)
- **Target SDK**: Android 14+ (API 34+)
- **Devices**: Phones & Tablets

### Windows
- **Minimum OS**: Windows 10 (Build 19041)
- **Recommended**: Windows 11
- **Devices**: Laptops & Desktops

---

## 🛠️ Technology Stack

### 📱 Android
- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Architecture**: MVVM + Clean Architecture
- **Database**: Room
- **Dependency Injection**: Hilt
- **Async**: Coroutines
- **VPN**: Android VpnService API

### 🖥️ Windows
- **Language**: C#
- **Framework**: .NET 8
- **UI Framework**: WinUI 3
- **Architecture**: MVVM
- **UI Components**: Fluent Design System

## 🚀 Quick Start

### Android
```bash
cd android
# Open in Android Studio
# File > Open > Select this directory
./gradlew build
./gradlew installDebug
```

### Windows
```bash
cd windows
# Open in Visual Studio 2022
# File > Open Project/Solution > Select OseiVPN.Windows.sln
# Build > Build Solution
# Debug > Start Debugging
```
